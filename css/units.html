CSS Units

Absolute Units

Always the same in any context

  • px - pixels (this is an example of 20 pixels)
  • in - inch
  • cm - centimeter

Relative Units

Change in any context

  • em - font size of an element (example: font-size is 10px so 1em is 10px and 2em is 20px)
    this is a box width of 10em and font-size of 10 pixels
  • rem - font size of the root element, either :root or html (reccomended)
    this is a box width of 10rem and font-size of 10 pixels

Viewport units

relate to size of viewport

  • vh - view height, 1vh = 1%This is an example of 5vh
  • vw - view width 1vw = 1% This is an example of 5vw
