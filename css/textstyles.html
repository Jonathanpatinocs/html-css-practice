Text Styles

Fonts

  • font-family - change font ex font-family:'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif , Anything after first font are fallback fonts
  • @font-face - link font url
    @font-face {
    font-family: my-cool-font;
    src: url(../fonts/the-font-file.woff);
    }
    h1 {
    font-family: my-cool-font, sans-serif;
    }

font-style

  • (html) em - Emphasis
  • (html) i - Italics or (css) font-style: italic:
  • letter-spacing - change space between words ex: letter spacing is 0.5em
  • line-height - space between lines in wrapped text, ex:
    The line
    height
    is 2em
  • text-transform - change case of given text ex: text-transfrom: uppercase
  • text-shadow - add shadow around text ex: text-shadow: 1px 1px 2px pink;, offset-x| offset-y| blur-radius| color
